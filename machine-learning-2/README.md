## Create Virtualenv
```
$ virtualenv machine-learning-2
```

## Activate virtualenv
`$ source bin/activate`


## Deactivate virtualenv
`$ deactivate`


## Install requirement libraries
```
$ pip install scipy
$ pip install numpy
$ pip install matplotlib
$ pip install pandas
$ pip install sklearn

OR

$ pip install -r requirement.txt
```


## Check requirement
`$ python requirement.py`
